# Semana 05 - Semáforo Inteligente

## O que foi feito
Desenvolvimento de um simulador de semáforo inteligente utilizando o aplicativo OctoStudio no celular. O objetivo foi programar um sistema que altera seu comportamento automaticamente com base em condições lógicas.

## Conceitos Principais Abordados
- **Condicionais (Se / Senão):** Uso de estruturas lógicas para verificar se uma condição é verdadeira (ex: se o tempo acabou, mude a cor do sinal).
- **Variável de Estado:** Criação de uma lógica para controlar em qual fase o semáforo está (Verde, Amarelo ou Vermelho) e garantir que ele siga a sequência correta.
