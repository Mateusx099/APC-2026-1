# Semana 11 - Exercícios com Microdados do PDAD 2024

## O que foi feito
Desenvolvimento de um script para extração, filtragem e análise de dados da Pesquisa Distrital por Amostra de Domicílios (PDAD 2024) do Distrito Federal, utilizando o arquivo oficial `moradores.csv`.

## Conceitos Aplicados
- Importação e estruturação de dados com a biblioteca Pandas.
- Tratamento manual de códigos de ausência de dados e recusa (99999 e 88888).
- Uso de laços de repetição loops para o cálculo manual de médias aritméticas sem o uso de funções embutidas de agregação de pacotes externos.
