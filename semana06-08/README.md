# Semanas 06 a 08 - Simulador de Circuito Arduino (Tinkercad)

## O que foi feito
Este bloco de semanas foi dedicado à transição das ferramentas de blocos no celular para o desenvolvimento em hardware virtual utilizando o Tinkercad Circuitos. Construí e programei um simulador de circuito utilizando a placa Arduino Uno, evoluindo de um pisca-pisca básico até a integração de sensores para controle de saída física.

## Conceitos Aprendidos
- **Estrutura do Arduino:** Entendimento das funções principais `void setup()` (configuração dos pinos) e `void loop()` (laço de repetição infinito que mantém o circuito rodando).
- **Pinos Digitais e Analógicos:** Configuração de pinos como saída (`OUTPUT`) para enviar energia a um LED e pinos de entrada (`INPUT`) para ler dados do ambiente.
- **Sensores e Controle Dinâmico:** Leitura de componentes analógicos (como um potenciômetro) usando `analogRead()` e aplicando esse valor para alterar o tempo de espera (`delay()`) do circuito em tempo real.
