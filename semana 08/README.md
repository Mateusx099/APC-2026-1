# Semana 08 - Funções e Biblioteca Reutilizável

## O que foi feito
Nesta semana criámos a nossa primeira biblioteca de funções em Python, o arquivo `lib_matematica.py`. A ideia foi agrupar várias rotinas de cálculo que usámos nas semanas anteriores (como médias e testes de números primos) num sítio só para podermos reutilizar o código mais facilmente nas próximas aulas.

## Conceitos aprendidos
- **def:** Comando usado para criar as funções em Python.
- **Docstrings:** Textos curtos dentro das funções que servem para explicar o que a função faz e o que ela precisa de receber para funcionar.
- **Escopo de variáveis:** Entendi que variáveis criadas dentro de uma função só funcionam lá dentro.
