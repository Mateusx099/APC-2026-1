Este simulador foi desenvolvido no octoStudio para demonstrar os conceitos fundamentais de computação: entrada, processamento e saída.


1. O sistema aguarda uma interação do usuário (Evento).
2. Solicita uma string de comando (Entrada de Dados).
3. Avalia se o comando é "girar" ou outro movimento (Estrutura Condicional).
4. Realiza a ação física no sprite (Sequência de Passos).
5. Exibe uma mensagem confirmando a execução (Saída de Dados).

este projeto exemplifica a programação antes da sintaxe
