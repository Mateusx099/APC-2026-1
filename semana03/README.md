# Semana 03 - Calculadora Interativa

## O que foi feito
Desenvolvimento de uma calculadora interativa utilizando o aplicativo OctoStudio no celular. O objetivo foi criar um sistema onde o usuário consegue interagir com elementos na tela para realizar operações matemáticas básicas.

## Conceitos Principais Abordados
- **Variáveis:** Uso de espaços na memória para guardar os números digitados e o resultado final.
- **Operadores Matemáticos:** Aplicação prática de soma, subtração, multiplicação e divisão dentro da lógica dos blocos.
- **Tipos de Dados:** Entendimento de como o sistema diferencia um texto (string) de um número para poder realizar os cálculos corretamente.
